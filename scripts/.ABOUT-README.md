# O que é a pasta scripts?
Ela guarda todos seus scripts (originalmente pensado em python, mas use o que lhe agradar) para o executável principal ter mais facilidade para executar os scripts. É a mesma lógica da pasta `shortcuts`. Basta ler o `shortcuts\.none` para saber mais.

Deixei alguns scripts que eu uso (em Windows) para os meus workflows. Não sei se vou criar uma branch só pra isso.
